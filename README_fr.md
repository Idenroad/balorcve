# BalorCVE
BalorCVE est un gestionnaire de vulnérabilités CVE (Common Vulnerabilities project) est avant tout fait pour être intégré au framework [Balor](https://github.com/idenroad/Balor)

## Fonctionnalités principales

- Recherche et consultation des CVE en mode offline (avec base de données l- Téléchargement et import des fichiers JSON officiels du NVD.
- Filtrage par mots-cle9s, dates, score CVSS et se9ve9rite9.
- Interface console interactive avec affichage enrichi (tableaux, panneaux).
- Support multilingue français/anglais avec détection automatique de la langue 
- Export des détails CVE au format HTML.

## Installation

```bash
pip install balorcve
```

## Utilisation

Lancez l'outil en console :
```bash
balorcve
```
Suivez les menus pour naviguer entre les modes offline et online.

BalorCVE est conçu pour s'intégrer facilement dans le framework Balor.
